Breve História da Web

Em 1989 Tim Berners-Lee, baseando-se no protocolo TCP/IP e na utilização de hipertextos (ou seja o HTML), criou a World Wide Web (WWW), primeiro navegador com o objetivo de globalizar a comunicação via Internet.

A primeira página da web foi publicada em 6 de agosto de 1991, podendo ser acessada até hoje pelo link: info.cern.ch/.

Sobre o HTML

(voltar)

HTML é a sigla para Hypertext Markup Language, ou em português,"Linguagem de Marcação de Hipertexto" e é utilizado para criação de documentos que podem ser interpretados por navegadores web.

É importante salientar que HTML não é uma linguagem de programação, e sim um arquivo de texto com delimitadores e sintaxe própria. Cada delimitador tem função e comportamento específicos que interferem em como o computador irá interpretar as informações contidas no arquivo de texto.

Estrutura Básica do HTML

(voltar)

Todo HTML é estruturado baseado em "Tags" (delimitadores que definem como o documento será interpretado pela página web). Para escrever um arquivo HTML podemos utilizar qualquer formatador de texto ou IDE. No meu caso utilizei o Visual Studio Code (VSCode). Clique aqui para baixar.

Um documento HTML consiste em três partes principais:

  1. "html" - Define o que será a página HTML
  2. "head" - O que será pré-carregado na página web
  3. "body" - O que irá aparecer para o usuário

Tags

(voltar)

As tags principais são as mencionadas acima:"html", "head" e "body" que formam a base para o arquivo HTML. Porém podemos utilizar diversas tags dentro de cada uma delas que irão ditar como o arquivo será interpretado quando aberto na página web.

Existem inúmeras tags, cada uma com especificicações e funcionalidades diferentes. Podemos conferir a lista atualizada no site da W3Schools, que é a empresa responsável por atualizar e definir os padrões globais para a sintaxe do HTML. Clique aqui para conferir a lista de tags HTML da W3Schools.

Atributos

(voltar)

São propriedades que as tags podem possuir. Elas podem ser Gerais (podem ser utilizadas por qualquer tag) ou Específicas (funcionam em apenas certas tags).

São exemplos de tags gerais:

  • id - Identificador de elemento
  • style - Utilizado para colocar comandos CSS para formatação
  • class - Utilizado para colocar comandos CSS para formatação de forma padronizada

São exemplos de tags expecíficas:

  • input type - Insere um campo de texto/número/cor
  • img src - Insere uma imagem por um caminho especificado
  • img width - Insere uma imagem por um caminho especificado com uma largura determinada
